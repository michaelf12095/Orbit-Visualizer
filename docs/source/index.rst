Orbit Visualizer documentation
==============================

This is an orbit visualizer I created myself that takes in the 5 classic Keplerian orbital elements as input for the "orbits" class. Those elements are:  
a -> semi-major axis  
e -> eccentricity  
inc -> inclination in degrees  
node -> longitude of ascending node in degrees  
arg -> argument of periapsis in degrees  
  
The visualizer specializes in elliptical orbits; however, it does have the capabilities to model hyperbolic trajectories (e>1). If you model a hyperbolic trajectory, please note that you will not get a completely accurate representation.
